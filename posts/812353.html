GitHub上传文件的方法

本文最后更新于：2 个月前

两种上传方法：

通过 git shell 命令上传项目到github

方法一：使用git工具把本地代码上传（更新）到github的仓库。

准备工作

  1. 先在 github官网 注册账号
  2. 再下载安装git工具

 

登录github

  1. 新建一个 repository(仓库)
  2. 填写仓库信息
  3. 第一次创建仓库会出现如下图所示信息。复制仓库地址以备用，其他不用管
  4. 或者创建仓库后，返回你的仓库（yours repositories），复制仓库地址以备用也是可以的~~

本地操作 关键步骤

  1. 找到你要上传的本地项目文件
    （注意：在本地建一个文件夹用作本地磁盘的仓库，以后将要上传的项目都存在本地仓库并进行与github的上传更新）
    如我的：D:\githubWorkSpace
  2. 右键点击要上传项目，如 text1。如果已安装好 git，则如图选择Git Bash Here
  3. 弹出命令窗口如图
  4. 输入如下代码git clone https://github.com/qwy8023/pythonDemo.git
    将你在 github 上新建的仓库克隆到本地。 （https://。。。。你之前复制的仓库地址）
  5. 此时，打开本地文件，除项目文件外会发现多出一个文件夹pythonDemo，然后将你的项目的所有文件都剪贴到该pythonDemo文件里面。
    我的项目代码文件是 text1.py
  6. 再次输入命令：
    cd pythonDemo ：进入到该文件夹
    git add . ：添加 pythonDemo 文件夹下的所有文件。（注：别忘记后面的 . 以及前面有个空格）

7.继续输入以下代码
git commit -m "first commit"：提交信息。（注：“first commit”里面换成你需要的描述，如“第一次提交”）
git push -u origin master ：把本地仓库push到github上面。（注：此步骤需要你输入帐号和密码进行登录）

8. 最后结果显示如图，命令行 上传项目操作完毕~~

github上查看

登录你的github，打开你的 pythonDemo 仓库，发现代码上传成功~~

通过 GitHub Desktop客户端上传项目到github

方法二：使用GitHub Desktop 应用把本地代码上传（更新）到github的仓库。
准备工作
下载安装 GitHub Desktop ，下载地址 https://desktop.github.com/
用 github 账号和密码登录GitHub Desktop
首次登录出现以下界面：

新建本地仓库并 push 到github上
第一步：
1.点击菜单栏左上角File 选择 New repository ，创建一个新的仓库到当地

2.在当地文件夹查看

3.将要上传的项目程序拷入

4.在GitHub Desktop显示如下

5.填写 summary(必填)，和相关描述并提交

6.将仓库push 到github 上。可点击菜单栏的repository 选择push ，也可以直接点击publish repository .
